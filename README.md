# 橋梁点検 写真野帳アプリ v2

## v2での修正点
- GitHub Pages向けに `.nojekyll` を追加
- サービスワーカーを一旦停止し、古いキャッシュによる不具合を回避
- `app.js?v=2` / `style.css?v=2` としてキャッシュ更新
- 画面上部に読込状態を表示

## アップロード方法
ZIPを解凍し、中身をリポジトリ直下に上書きアップロードしてください。
`index.html` がリポジトリ直下にある状態にしてください。

## 注意
初期版は PDF.js と JSZip をCDNから読み込みます。社内ネットワークでCDNがブロックされると、PDF読込やZIP出力が動きません。
